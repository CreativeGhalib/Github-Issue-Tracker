1️⃣ What is the difference between var, let, and const?

var is the old way to declare variables, but it can be re-declared and updated. let are both block-scoped, so they could use mutation - but const values are bound at time of declaration and cannot change afterwards in the same scope.

2️⃣ What is the spread operator (...)?

The spread operator is mainly used to expand values from an array or object. It is commonly used for copying, merging and combining array and object items in a cleaner way.

3️⃣ What is the difference between map(), filter(), and forEach()?

map() creates a new array by transforming each item. filter() and forEach() respectively produce arrays containing only those entries for which a given requirement holds true; The latter method does not return anything back to the user base.

4️⃣ What is an arrow function?

An arrow function is a shorter way to write a function in JavaScript with =>, used for its cleanliness and is often used in small functions and callbacks.
